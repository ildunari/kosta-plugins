# Education Writing -- Domain-Specific Gates and Conventions

## Document Types and Their Structures

### Curriculum Document

```
1. Program Overview
   1.1 Mission and Philosophy
   1.2 Target Learners
   1.3 Program Goals
2. Course Sequence / Scope and Sequence
3. Individual Course Descriptions
   3.1 Course Title and Code
   3.2 Prerequisites
   3.3 Course Description
   3.4 Learning Objectives / Outcomes
   3.5 Content Outline (topics by week/unit)
   3.6 Required Texts and Materials
   3.7 Assessment Plan
   3.8 Alignment Map (objectives to assessments to standards)
4. Assessment Strategy
   4.1 Formative Assessments
   4.2 Summative Assessments
   4.3 Rubrics
5. Instructional Methods
6. Accessibility and Accommodations
7. Program Assessment and Continuous Improvement
Appendices (Sample Syllabi, Rubrics, Alignment Matrices)
```

Key conventions:
- Every learning objective must be measurable and align to at least one assessment
- Content coverage must be traceable to program goals
- Prerequisites must form a coherent progression (no circular dependencies)

### Assessment Rubric

```
Task/Assignment Description
Learning Objectives Assessed
Performance Levels (e.g., Exemplary / Proficient / Developing / Beginning)
Criteria
  Criterion 1: [Description]
    Exemplary: [Observable behaviors]
    Proficient: [Observable behaviors]
    Developing: [Observable behaviors]
    Beginning: [Observable behaviors]
  Criterion 2: ...
Scoring Guide (point allocation)
```

Key conventions:
- Each performance level must describe observable, distinguishable behaviors
- Adjacent performance levels should differ by clear, identifiable criteria (not just "better" or "more")
- Point allocations should reflect the relative importance of criteria
- Rubric criteria must align with the stated learning objectives

### Accreditation Report / Self-Study

```
Executive Summary
1. Institutional Overview
2. Standard-by-Standard Narrative
   Standard 1: [Standard Name]
     Description of Compliance
     Evidence
     Analysis of Strengths and Challenges
     Action Plan for Improvement
   Standard 2: ...
3. Student Achievement Data
   3.1 Retention and Graduation Rates
   3.2 Assessment Results
   3.3 Post-Graduation Outcomes
4. Continuous Improvement Process
5. Resource Allocation
Appendices (Data Tables, Evidence Documents, Organizational Charts)
```

### Research Proposal (Educational)

```
1. Project Summary / Abstract
2. Project Description
   2.1 Significance
   2.2 Literature Review
   2.3 Theoretical Framework
   2.4 Research Questions / Hypotheses
   2.5 Methodology
     2.5.1 Research Design
     2.5.2 Participants and Sampling
     2.5.3 Instruments
     2.5.4 Data Collection Procedures
     2.5.5 Data Analysis Plan
   2.6 Timeline
   2.7 Limitations
3. Broader Impacts / Dissemination Plan
4. Budget and Justification
5. References
Appendices (Instruments, Consent Forms, IRB Approval)
```

## Data Verification Gates for Education Documents

### Learning Objectives Alignment

The core quality check for any educational document: objectives, instruction, and assessment must align.

**Bloom's Taxonomy verification:**

Every learning objective should use a measurable action verb that indicates the cognitive level:

| Level | Example Verbs | Red-Flag Verbs |
|-------|--------------|----------------|
| Remember | List, define, identify, recall, name | -- |
| Understand | Explain, summarize, interpret, classify | "Understand" (not measurable) |
| Apply | Solve, demonstrate, use, implement | -- |
| Analyze | Compare, contrast, differentiate, examine | -- |
| Evaluate | Justify, critique, assess, judge | "Appreciate" (not measurable) |
| Create | Design, construct, develop, compose | "Know" (not measurable) |

**Verification gate:**
1. Every objective uses a measurable verb (not "understand," "know," "appreciate," "be familiar with")
2. Every objective specifies conditions and criteria when possible: "Given [conditions], the student will [verb] [content] to [standard]"
3. Every objective maps to at least one assessment that measures the stated verb at the stated cognitive level
4. Bloom's levels progress logically through the course (foundational before advanced)
5. The total number of objectives is realistic for the time allotment (rule of thumb: 3-5 objectives per credit hour, not 20)

### Assessment Validity

Assessments must measure what they claim to measure.

**Content validity:** Does the assessment cover the learning objectives it claims to measure?
- Map each assessment item or rubric criterion to a specific learning objective
- Flag objectives without any assessment coverage
- Flag assessment items that don't connect to any objective (assess without purpose)

**Construct validity:** Does the assessment measure at the right cognitive level?
- An objective that says "analyze" should not be assessed by a multiple-choice recall question
- Match the assessment method to the Bloom's level:

| Bloom's Level | Appropriate Assessment Methods |
|---------------|-------------------------------|
| Remember | Multiple choice, matching, fill-in-the-blank |
| Understand | Short answer, concept mapping, explain-in-own-words |
| Apply | Problem sets, case studies, simulations |
| Analyze | Compare/contrast essays, data analysis, case studies |
| Evaluate | Critiques, position papers, peer review |
| Create | Projects, designs, portfolios, research papers |

**Rubric quality:** For rubric-based assessments:
- Performance levels are evenly spaced in quality (no jumps from "excellent" to "failing")
- Descriptors are specific enough that two raters would assign the same score (inter-rater reliability)
- Criteria are independent (scoring one criterion should not determine the score of another)
- Point weights reflect stated priorities

### Accessibility -- Universal Design for Learning (UDL)

UDL guidelines ensure materials are accessible to learners with diverse needs.

**Three UDL principles:**

| Principle | Question to Ask | Common Failure |
|-----------|----------------|----------------|
| Multiple means of engagement | Can learners with different motivations and interests engage? | Single assessment format with no alternatives |
| Multiple means of representation | Can learners with different perceptual needs access the content? | Text-only materials with no visual or auditory alternatives |
| Multiple means of action and expression | Can learners with different physical and expressive abilities demonstrate learning? | Only timed written exams |

**Verification gate:**
1. Materials are available in or convertible to accessible formats (screen-reader compatible, captioned video, alt text for images)
2. Assessment options include at least one alternative modality for learners with documented accommodations
3. Time constraints are justified by learning objectives (if timing isn't the skill being measured, flexible timing should be available)
4. Color is not the sole means of conveying information in materials or rubrics
5. Technology requirements are stated and alternatives are provided for learners without access

### Institutional Format Compliance

Accreditation reports and institutional documents must follow the accreditor's format exactly.

**Verification gate:**
1. Every required standard or section from the accreditor's template is addressed
2. Evidence is tagged to the specific standard it supports
3. Data tables use the accreditor's required format and definitions
4. Student achievement metrics are calculated using the accreditor's definitions (e.g., retention = fall-to-fall, not fall-to-spring)
5. Self-study narrative length falls within accreditor guidelines
6. All required appendices are present and referenced in the narrative

## Value Presentation Conventions

- Achievement data: Report with denominators and timeframes: "78% of students (156 of 200) achieved proficiency on the final assessment (Spring 2025)"
- Retention rates: Specify the cohort, definition, and timeframe: "First-time, full-time freshmen entering Fall 2023: 82% returned Fall 2024"
- Grade distributions: Use tables with headings: Course | Section | N | % A | % B | % C | % D/F | % W
- Survey results: Report response rates alongside findings: "85% of graduates (n=340, 68% response rate) reported employment in their field within 6 months"
- Comparative data: Always include the comparison group and its source: "compared to the national average of 65% (NSSE, 2025)"
- Effect sizes: Report for educational interventions: "Cohen's d = 0.45, indicating a medium effect"
- Standard setting: Describe how cut scores were determined: "the proficiency cut score of 70% was established by modified Angoff method with 8 expert judges"

## Common Education Writing Pitfalls

| Pitfall | Example | Fix |
|---------|---------|-----|
| Unmeasurable objectives | "Students will understand the Civil War" | "Students will analyze three causes of the Civil War and evaluate their relative significance" |
| Bloom's mismatch | Objective says "evaluate," assessment is a recall quiz | Match assessment type to cognitive level |
| Missing alignment | Objective exists but no assessment measures it | Map every objective to an assessment; fill gaps |
| Rubric vagueness | "Good work" as a performance descriptor | "Identifies 3-4 relevant sources and synthesizes findings into a coherent argument" |
| Data without context | "Our retention rate is 76%" | "Our retention rate is 76%, compared to 72% for peer institutions and 68% nationally (IPEDS, 2024)" |
| Grade inflation mask | "95% of students earned A or B" presented as success | Include assessment validity data; compare to standardized benchmarks |
| Accessibility afterthought | "Accommodations available upon request" with no proactive design | Build UDL into the design; accommodations supplement, not replace, accessible design |
| Curriculum bloat | 47 learning objectives for a 3-credit course | 12-15 well-crafted objectives that are actually assessed |

## Compliance Checklists

### Curriculum Document

- [ ] Every learning objective uses a measurable action verb
- [ ] Every objective aligns to at least one assessment
- [ ] Every assessment aligns to at least one objective
- [ ] Prerequisites form a coherent progression with no circular dependencies
- [ ] Content outline covers all objectives
- [ ] Bloom's levels progress logically through the course
- [ ] Instructional methods are appropriate for the stated cognitive levels
- [ ] Accessibility considerations are addressed (UDL principles)
- [ ] Required textbooks and materials are current and available

### Assessment Rubric

- [ ] Every criterion aligns to a stated learning objective
- [ ] Performance levels are distinguishable by observable behaviors
- [ ] Adjacent levels differ by specific, identifiable criteria
- [ ] Point allocations reflect stated priorities
- [ ] Rubric can achieve acceptable inter-rater reliability
- [ ] Instructions to students are clear and complete
- [ ] Rubric does not inadvertently penalize language or cultural differences unrelated to the objective

### Accreditation Self-Study

- [ ] Every required standard is addressed with narrative and evidence
- [ ] Data tables use the accreditor's definitions and formats
- [ ] Student achievement metrics are calculated correctly
- [ ] Evidence is tagged to specific standards
- [ ] Strengths and challenges are analyzed honestly (not just strengths)
- [ ] Action plans are specific, measurable, and time-bound
- [ ] Narrative length falls within guidelines
- [ ] All appendices are present, referenced, and current

### Research Proposal

- [ ] Research questions are clearly stated and answerable
- [ ] Methodology is appropriate for the research questions
- [ ] Sample size is justified (power analysis for quantitative; saturation argument for qualitative)
- [ ] Instruments are described with reliability and validity evidence
- [ ] Data analysis plan matches the research design
- [ ] IRB considerations are addressed
- [ ] Timeline is realistic for the proposed scope
- [ ] Budget items are justified and tied to project activities
- [ ] Literature review establishes the gap the study addresses
- [ ] Limitations are acknowledged proactively
