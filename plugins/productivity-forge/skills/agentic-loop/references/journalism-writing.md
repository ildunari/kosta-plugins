# Journalism Writing -- Domain-Specific Gates and Conventions

## Document Types and Their Structures

### Investigative Report

```
Headline (specific, factual, not clickbait)
Byline and Date
Lede (the finding, in one paragraph)
Nut graf (why this matters, broader context)
Evidence Section 1
  Key finding
  Supporting documents/data
  Source quotes (on the record)
Evidence Section 2
  Key finding
  Supporting documents/data
  Response from subject of investigation
Evidence Section N ...
Context / Background
Methodology (how the investigation was conducted)
What Happens Next / Implications
Editor's Note (if applicable: disclosure of conflicts, correction history)
```

Key conventions:
- The lede states the most important finding -- do not bury it
- Every factual claim must be supported by documents, data, or named sources
- The subject of the investigation must be given an opportunity to respond, and the response (or refusal to respond) must be included
- Methodology section explains how documents were obtained, how data was analyzed, and what verification steps were taken

### Fact-Check Article

```
Claim (exact quote or paraphrase with attribution)
Verdict / Rating (True / Mostly True / Half True / Mostly False / False / Unverifiable)
Context (when and where the claim was made)
Analysis
  What the claim states
  What the evidence shows
  Key sources and their credibility
  Nuances and caveats
Sources List (with links where available)
Correction History (if the rating or analysis has been updated)
```

Key conventions:
- The exact claim being checked must be quoted or precisely paraphrased
- The verdict must be supported by the analysis, not vice versa (don't start with the verdict and work backward)
- Sources should be primary where possible (the dataset itself, the law itself, the transcript itself)
- If reasonable people could interpret the evidence differently, acknowledge it

### Data Journalism Piece

```
Headline
Byline and Date
Summary (key finding in plain language)
Narrative (story told through data)
  Context
  Data presentation (charts, maps, tables)
  Analysis and findings
  Limitations
Methodology Box
  Data sources
  Analysis methods
  Code/data availability
  Known limitations
Expert Commentary / Response
```

Key conventions:
- Every chart must have a title, axis labels, source citation, and date range
- Methodology must be transparent enough for another journalist to reproduce the analysis
- Data should be available for download or the refusal to share should be explained
- Statistical findings should be presented in plain language alongside technical notation

### Breaking News / Wire Story

```
Headline (subject-verb-object, present tense)
Dateline (CITY -- )
Lede (who, what, when, where)
Second paragraph (why, how, attribution)
Background paragraphs (expanding detail in decreasing importance)
Reaction / Response
Context
```

Key conventions:
- Inverted pyramid: most important information first, least important last
- Every paragraph should stand alone -- editors cut from the bottom
- Attribution for every factual claim that isn't directly observed by the reporter
- Update with corrections and new information, timestamped

## Data Verification Gates for Journalism Documents

### Source Verification (The Two-Source Rule)

The foundational journalism standard: any factual claim that is not a matter of public record requires at least two independent sources.

**Verification gate:**
1. For each key factual claim, identify all supporting sources
2. Verify that sources are genuinely independent (not two people who heard the same rumor)
3. Sources with firsthand knowledge are stronger than secondhand accounts
4. Documentary evidence (documents, records, data) is stronger than testimonial evidence
5. On-the-record sources are preferred; if anonymous sources are used, the reason for anonymity must be stated and the editorial decision to grant it must be justified

**Source strength hierarchy:**
| Source Type | Strength | Use |
|-------------|----------|-----|
| Official public records (court filings, SEC filings, government databases) | Highest | Can stand alone as a source |
| Primary documents (emails, contracts, internal memos) with verified provenance | High | Can stand alone if provenance is confirmed |
| On-the-record named sources with firsthand knowledge | High | Strongest testimonial evidence |
| On-the-record named sources with secondhand knowledge | Medium | Needs corroboration |
| Anonymous sources with firsthand knowledge | Medium | Needs corroboration; justify anonymity |
| Anonymous sources with secondhand knowledge | Low | Use only with strong corroboration |
| Social media posts | Variable | Verify identity of poster; screenshot for preservation; do not treat as fact without corroboration |

### Quote Accuracy

Quotes must be exact. Paraphrases must be accurate representations of what was said.

**Verification gate:**
1. Every direct quote (in quotation marks) must be verbatim from a recording, transcript, or contemporaneous notes
2. If a quote is cleaned up (removing "ums" or false starts), this must follow the publication's style guide and must not change meaning
3. Partial quotes must not misrepresent the full statement through selective extraction
4. Quotes translated from another language should note this: `"The situation is unacceptable," he said, speaking through a translator` or `[translated from Spanish]`
5. Ellipses in quotes must not change the meaning of the original statement
6. When quoting from written documents, verify against the original document, not a secondary source's transcription

**Quote attribution rules:**
- Every quote has an attribution: who said it, when, and in what context
- "Said" is the default attribution verb (not "claimed," "admitted," "revealed" -- these imply judgment)
- Attribution verbs that imply judgment are reserved for when the judgment is supported by evidence

### Defamation Risk

Journalism must balance public interest reporting with legal risk. Defamation occurs when a false statement of fact causes harm to someone's reputation.

**Pre-publication checklist:**
1. Is the statement presented as fact or opinion? (Opinion is generally protected; fact must be provable)
2. Is the subject a public figure or private individual? (Public figures face a higher burden -- actual malice; private individuals only need negligence)
3. Can every factual claim be supported by evidence if challenged?
4. Has the subject been given a fair opportunity to respond?
5. Is the response included, even if it's "declined to comment"?
6. Are there any implications (innuendo, juxtaposition) that could be read as unstated factual claims?
7. Have all sources of potential ambiguity been eliminated through precise language?

**High-risk patterns:**
| Pattern | Risk | Mitigation |
|---------|------|------------|
| Implying guilt by association | "CEO was seen with convicted fraudster" | Only include if newsworthy and contextualized |
| Headline broader than story | "Company Defrauds Customers" when story shows billing errors | Headline must be supported by the story's evidence |
| Unnamed accusers | "Multiple sources say he committed fraud" | Name sources when possible; if anonymous, explain why and ensure the underlying facts are verifiable |
| Pre-conviction language | "The criminal" or "the fraudster" before conviction | "The defendant" or "the accused" or "the person charged with" |

### Editorial Standards

**Corrections protocol:**
1. Errors of fact must be corrected immediately upon discovery
2. Corrections must be clearly labeled and visible (not buried)
3. The original error and the corrected information should both be stated
4. The date and time of the correction must be noted
5. For online articles, corrections are appended to the article; the article is not silently edited

**Conflicts of interest:**
1. Financial interests in the subject of reporting must be disclosed
2. Personal relationships with sources must be disclosed to editors
3. Gifts from sources above a nominal value must be refused or disclosed
4. Previous employment by the subject of investigation must be disclosed

**Independence:**
1. Sources do not have approval rights over the story
2. Advertisers do not influence editorial content
3. The subject of an investigation does not get to see the full story before publication (they get their specific allegations for response)

## Value Presentation Conventions

- Numbers: Spell out one through nine; use numerals for 10 and above. Exception: ages, percentages, and data are always numerals
- Dates: Month Day, Year: "March 28, 2026" (AP style)
- Times: Lowercase a.m. and p.m.: "3:30 p.m. ET"
- Money: Dollar sign with numerals: "$5 million" not "five million dollars"
- Percentages: Numeral with "percent" (not %): "15 percent" (AP style) or "15%" (varies by publication)
- Attribution: Present tense for ongoing conditions ("says"), past tense for specific past statements ("said")
- Titles: Before name, capitalized and not abbreviated: "President Biden said"; after name, lowercase: "Joe Biden, president of the United States"
- Ages: Always use numerals: "the 45-year-old defendant"
- Data visualization: Every chart needs title, source, date range, and axis labels

## Common Journalism Writing Pitfalls

| Pitfall | Example | Fix |
|---------|---------|-----|
| Single-source claim | Major allegation from one anonymous source | Corroborate with a second independent source or documentary evidence |
| Buried lede | Three paragraphs of context before stating the finding | Lead with the finding; provide context after |
| False balance | Giving equal weight to a scientist and a non-expert on a scientific consensus | Weight coverage by the strength of the evidence, not the number of voices |
| Unverified social media | Reporting tweets as factual claims | Verify the poster's identity and the claim's accuracy independently |
| Missing response | Alleging wrongdoing without contacting the subject | Always contact the subject and include their response or note that they declined |
| Selective quoting | Using half a sentence that reverses the speaker's meaning | Quote in full context; include enough surrounding material to preserve meaning |
| Correlation headline | "Study Links Coffee to Cancer" when the study found a weak correlation | Report the actual finding with appropriate hedging; include effect size |
| Outdated data | "Crime rates are rising" using data that's 3 years old | Use the most recent available data; state the time period |

## Compliance Checklists

### Investigative Report

- [ ] Every factual claim is supported by at least two independent sources or documentary evidence
- [ ] Subject of investigation was given opportunity to respond
- [ ] Response (or refusal to respond) is included in the story
- [ ] All quotes are verified against recordings, transcripts, or contemporaneous notes
- [ ] Methodology is transparent and described in the story
- [ ] Defamation review completed (all factual claims provable if challenged)
- [ ] Conflicts of interest disclosed
- [ ] Documents are authenticated (provenance verified)
- [ ] Statistical claims are accurate and contextualized
- [ ] Pre-publication legal review completed (for high-risk stories)

### Fact-Check Article

- [ ] Exact claim is quoted with attribution (who said it, when, where)
- [ ] Primary sources consulted (not just other news reports)
- [ ] Verdict is supported by the analysis (not the other way around)
- [ ] Nuances and caveats are acknowledged
- [ ] Rating scale is applied consistently with previous fact-checks
- [ ] Sources are listed with links
- [ ] Correction history is noted if the analysis has been updated
- [ ] The strongest counterargument to the verdict is addressed

### Data Journalism

- [ ] Data sources are identified with download links or FOIA references
- [ ] Methodology is described in enough detail for reproduction
- [ ] Statistical methods are appropriate for the data type and question
- [ ] Visualizations have titles, axis labels, source citations, and date ranges
- [ ] Limitations are stated (missing data, definitional changes, geographic coverage)
- [ ] Expert commentary provides context for the findings
- [ ] Code is available for review (or the refusal to share is explained)
- [ ] Outliers and anomalies are investigated, not just reported
- [ ] Plain language summary accompanies technical findings
