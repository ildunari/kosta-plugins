## Tracking Goals

Over time, this log reveals:
- Which interview questions actually affect the output (drop the ones that don't)
- Whether XcodeGen vs raw xcodeproj matters for beginners (assumption to test)
- Whether the Phase 7 first-build step catches real issues or just slows things down
- Whether the generated CLAUDE.md is useful in subsequent sessions
- Which starter templates get modified immediately (sign they're wrong)
